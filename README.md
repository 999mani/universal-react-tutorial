# Universal React Tutorial

A tutorial on building a universal web app with React.

### Install

```sh
$ git clone https://github.com/remarkablemark/universal-react-tutorial.git
$ cd universal-react-tutorial
$ npm install
```

### Run

```sh
$ npm start
```
